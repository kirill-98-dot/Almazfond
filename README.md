изменения по Almazfond

Краткий список изменений по SEO

1. Починил редиректы. Выглядит так: 
https://www.almazfond.ru/ -> 301 -> https://almazfond.ru/
http://www.almazfond.ru/  -> 301 -> https://almazfond.ru/

2. Страницы, которые не нужно продвегать в seo(cart, catalog, shop, Checkout и My account) убрал из индексации. 
Они засоряли sitemap. Хотел спросить, нужно ли двигать /cookie-policy/, /privacy-policy/ и /processing-personal-data/?
И если они нужны только юридически, надо бы поставить им noindex, как корзине

3. Желательно исправить title h1 и description на более понятные: 
c: title: Категории - almazfond
description: отсутствует
H1: Искусство веков: шедевры Златоуста
OG title: Категории - almazfond
OG description: отсутствует
на: 
Title:
Каталог элитных подарков из Златоуста - Almazfond
Description:
Каталог премиальных подарков ручной работы: сувенирное оружие, бизнес-сувениры, иконы, восточные подарки, посуда и интерьерный декор из Златоуста.
H1:
Каталог элитных подарков

4. Были убраны мусорные WP дефолты из siteMap: 
https://almazfond.ru/privet-mir/
https://almazfond.ru/category/bez-rubriki/

5. В категориях поменял Seo-заголовок и описание на более понятнее, то есть теперь страница в коде отдает это:
Title: Каталог элитных подарков из Златоуста - Almazfond
Description: Каталог премиальных подарков ручной работы: сувенирное оружие, бизнес-сувениры, иконы, восточные подарки и интерьерный декор.
Canonical: https://almazfond.ru/categories/
Status: 200

6. Проиверил, что карточки отдают Product

7. Проверил микроразметку. Так же включил BreadcrumbList, отвечающий за разметка, которая объясняет поисковикам структуру страницы и место текущей страницы в иерархии сайта.
То есть теперь товарная страница выдает: 
Organization
WebSite
BreadcrumbList
Product
Offer

8. Убрал article, он не нужен для товарной странице. Он обычно служит для  статей: блог, новости, гайды, обзоры, инструкции.

9. Изменил описания для шаблонов описания товаров в Rank Math. Они будут работать массово и не потребуют редактировать 500+ карточек вручную.

10. Поменял шаблоны для категорий товаров на более универсальные и красивые

11. Прописал индивидуальные Seo title/description и фокусный ключ для 8 категорий.

12. Проверил 8 основных категорий на публичной части сайта.
Все отдают:
Status: 200
Robots: index, follow
Canonical: на саму категорию
Кодировка: UTF-8
Title и Description: отдаются в HTML

13. Проверил товары по одному из каждой основной категории.
Товарные карточки отдают:
Status: 200
Canonical: корректный
H1: есть
Product schema: есть
